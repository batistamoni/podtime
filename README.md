# PodTime 🎧

Aplicación web para organizar, guardar y seguir el progreso de los podcasts que escuchas.

---

## Sobre el proyecto

Muchos oyentes de podcasts no tienen una forma sencilla de:

- recordar en qué episodio se quedaron
- guardar podcasts interesantes
- seguir su progreso de escucha

**PodTime** nace como una plataforma simple para gestionar todo eso en un solo lugar.

Este proyecto forma parte de mi aprendizaje como desarrolladora web y será mi **proyecto final del ciclo DAW (Desarrollo de Aplicaciones Web)**.
